\# AI-Powered Fake Instagram Business Account Detection System



AI-powered web platform for detecting fake Instagram business accounts using machine learning and engagement analysis.



\## Features

\- Fake account detection

\- Engagement analysis

\- Real-time account checking

\- Trust score generation

\- Web scraping support



\## Technologies Used

\- Python

\- Flask

\- Machine Learning

\- HTML

\- CSS

\- JavaScript

\- Bootstrap



\## Project Goal

This project helps users identify fake or suspicious Instagram business accounts to improve trust and reduce scams on social media platforms.
